Props কী?
The Basics

React-এ ডেটা পাঠানোর সবচেয়ে সহজ উপায় — function parameter-এর মতোই!

JS
Props = Function Parameter
Parent → Child flow
Read-only (immutable)
Code Example
Greeting.jsx
1// ✦ Normal JS — একটি সাধারণ function
2function greet(name) {
3  return `Hello, ${name}!`;
4}
5
6// ✦ React Component — props IS সেই parameter
7function Greeting(props) {
8  return (
9    <h1>Hello, {props.name}!</h1>
10  );
11}
12
13// ✦ Component ব্যবহার করা
14<Greeting name="Arafat" />
15// ➜ Output: Hello, Arafat!
